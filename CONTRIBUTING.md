# Contributing

K is built n=1: the roadmap is its founder's life, and this repository is a fresh-history
mirror of the private working repo — pull requests cannot merge here in the usual way.

What is genuinely welcome:

- **Issues** — bug reports, security findings, and questions about the design. Read
  PRINCIPLES.md first; a lot of apparent bugs (silence, refusal to act, terse copy) are laws.
- **Forks** — the license (AGPL-3.0) exists so you can build your own K. Sovereignty means
  you run it; there is nothing to sign up for and no hosted version coming.
- **Design conversation** — the interesting surface is the eval thesis and the attention
  economics. Issues that engage those get engaged with.

Patches: open an issue with the diff or a fork link. If it fits, it gets reimplemented on the
private side (with credit) and flows back through the mirror.
